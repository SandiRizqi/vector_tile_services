Apa itu Vector Tiles?

Vector tiles adalah format data peta modern yang mengirimkan data geospasial dalam bentuk vektor (titik, garis, dan poligon) daripada gambar raster. Teknologi ini memungkinkan rendering peta yang lebih cepat, interaktif, dan dapat dikustomisasi dengan ukuran file yang lebih kecil.

Service ini menyediakan vector tiles dalam format Mapbox Vector Tile (MVT/PBF), yang merupakan standar industri untuk pemetaan web modern dengan performa tinggi.

Fitur Utama

Performa Tinggi

Dibangun dengan Actix Web dan Rust untuk kecepatan maksimal dan penggunaan resource yang efisien.

Styling Fleksibel

Data vektor dapat di-style secara dinamis di sisi klien sesuai kebutuhan aplikasi Anda.

Ukuran Optimal

Format PBF yang terkompresi menghasilkan ukuran transfer data minimal dengan kualitas maksimal.

Standar Industri

Kompatibel dengan Mapbox GL JS, MapLibre, Leaflet, dan library pemetaan modern lainnya.

Real-time Updates

Mendukung pembaruan data dinamis tanpa perlu regenerasi seluruh tileset.

Multi-platform

Dapat digunakan di web, mobile, dan desktop dengan performa konsisten.

Available Layers

Lihat daftar lengkap layer yang tersedia di service ini. Setiap layer memiliki konfigurasi dan properti yang berbeda sesuai dengan jenis data geospasial yang disediakan.

API Endpoints

GET /layers

Menampilkan daftar semua layer yang tersedia dalam format JSON.

GET /tiles/{table_name}/{z}/{x}/{y}.pbf

Parameters:
z - Zoom level (0-22)
x - Tile column coordinate
y - Tile row coordinate

Technology Stack

Actix Web Rust MVT/PBF Vector Tiles PostgreSQL/PostGIS

Cara Menggunakan

Integrasikan vector tiles service ini dengan library pemetaan favorit Anda seperti Mapbox GL JS atau MapLibre GL JS. Cukup arahkan tile source ke endpoint yang tersedia dengan parameter z/x/y yang sesuai.

Contoh implementasi: 

map.addSource('vector-tiles', {
  type: 'vector',
  tiles: ['http://localhost:8080/tiles/{table_name}{z}/{x}/{y}.pbf'],
  minzoom: 0,
  maxzoom: 14
});
